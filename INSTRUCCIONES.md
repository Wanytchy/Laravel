# Instrucciones para completar la documentación del Laboratorio #2

Este paquete incluye la estructura recomendada para tu entrega.  

## Carpetas
- `docs/capturas/`: Coloca aquí las **capturas de pantalla** de tu laboratorio (login, dashboard, etc.).  
- `database/backup/`: Guarda aquí el **respaldo SQL** de tu base de datos.  

## Pasos
1. Sustituye los archivos `*_placeholder.png` por las capturas reales.  
   - Ejemplo: `login.png`, `dashboard.png`.
2. Exporta tu base de datos desde MySQL/MariaDB (phpMyAdmin o consola) y guarda el `.sql` en `database/backup/`.  
   - Renombra el archivo con la fecha, ej.: `backup_lab_login_20250928.sql`.
3. Actualiza el archivo `README.md` con las rutas de las imágenes y el respaldo.
4. Sube todo el contenido a tu repositorio GitHub/GitLab.
